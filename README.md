# Proyecto: Consumo de API

## Descripción
Este proyecto consiste en una aplicación web tipo Pokédex que consume una API pública para mostrar información de diferentes Pokémon en tarjetas.

Elegí esta temática porque me gustan los videojuegos y quería hacer algo visual y diferente.

## API utilizada
Se utilizó la API pública PokeAPI:
https://pokeapi.co/

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Visualización de Pokémon en tarjetas (cards)
- Consumo de datos en tiempo real desde una API
- Buscador para filtrar Pokémon por nombre
- Diseño responsivo adaptable a diferentes pantallas
- Colores dinámicos según el tipo de Pokémon

## Cómo ejecutar
1. Descargar el proyecto
2. Abrir el archivo index.html en el navegador
